# Observação do site [ACBrasil](https://acbrasil.org.br/cms/)

## Header

### Problemas principais
-  Apresenta uma div que seria o menu chamada "[menu-associado]", mas só funciona na resolução de tela para celular.

## Footer

### Problemas principais
- O link de Home não funciona.

## [Página Inicial](https://acbrasil.org.br/cms/)
### Problemas principais
- Apresenta um pouco de lentidão ao passar os artigos que são expostos inicialmente.
- Links de redirecionamento para "Webinars", "Mais artigos", "Seja um Associado".
- Possui uma div de "[acb-newsletter]" que não possui compatibilidade em nenhum tipo de tela.

## [Quem somos](https://acbrasil.org.br/cms/quem-somos/)

### Problemas Principais
- O link "Faça sua associação" não funciona.
- As imagens dos fundadores são clicáveis mas não redirecionam para nenhum lugar.

## [O que fazemos](https://acbrasil.org.br/cms/o-que-fazemos/)

### Problemas principais
- O header sobrepõem o texto da página.
- O formulário não verifica se o número tem a quantidade correta de números.

## [Artigos](https://acbrasil.org.br/cms/c/artigos/)

### Problemas principais
- "Filtro" que altera a natureza da notícia não é padronizado e não aparece todos artigos postados.
- No final onde mostra as "Páginas" para navegar entre os artigos possui um distanciamento entre o texto que informa a página que está e o seletor de página.
- Quem é Webmasterbr.
- O link para enviar email para o Webmasterbr não tem a tag "_blank", redirecionando assim diretamente para outra página.
